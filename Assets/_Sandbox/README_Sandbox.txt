_Sandbox Folder Rule

- Use this folder for temporary experiments, candidate prefabs, candidate scenes, and temporary scripts.
- Do not treat files here as official gameplay assets.
- Move useful work into the official folders before connecting it to the main scene.
- Suggested promotion targets:
  Player scripts: Assets/Scripts/Player
  UI scripts and prefabs: Assets/Scripts/UI, Assets/Prefabs/UI
  Building, crafting, inventory, and combat systems: Assets/Scripts/Systems
  Reusable prefabs: Assets/Prefabs
  Shared visual assets: Assets/Arts, Assets/Materials, Assets/UI
- Move abandoned files to Assets/_Disabled or delete them after team review.
