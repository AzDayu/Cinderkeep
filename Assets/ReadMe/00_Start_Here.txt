현재 게임 코드 기준
- 기존 Play Our Game 구현은 Assets/_Disabled 아래로 이동했습니다.
- 새 게임 뼈대는 Assets/Scripts/Managers, Assets/Scripts/Data, Assets/Scripts/Models에 있습니다.
- GameManager만 싱글톤입니다.
- 다른 매니저는 Awake에서 흐름을 시작하지 않고 Initialize 함수로 준비합니다.
- 새 게임 스크립트는 장면 전체 검색 API를 사용하지 않습니다.

팀 작업 원칙
- 작업 전 본인 브랜치와 작업 범위를 확인합니다.
- 팀원이 만든 브랜치는 바로 덮어쓰지 말고 변경 의도를 먼저 읽습니다.
- 기능을 합치기 전에는 Main Workspace Room에서 동작 범위를 확인합니다.
- 커밋 전 변경 파일, 실행 가능 여부, 충돌 가능성을 확인합니다.
