팀 작업 흐름

1. Main_Lobby에서 본인 캐릭터를 선택합니다.
2. 개인 작업은 본인 Assets 폴더와 개인 작업 씬에서 먼저 실험합니다.
3. 공통으로 쓸 수 있는 기능만 Shared Workspace 또는 Main Workspace Room에 연결합니다.
4. 기능이 동작하면 팀장과 담당자가 확인할 수 있게 변경 내용을 짧게 남깁니다.
5. 확인된 작업만 커밋, 푸쉬, 머지 대상으로 올립니다.

권장 방식
- 개인 작업 공간: 실험, 연습, 개인 기능 초안
- 공동 작업 공간: 팀 프로젝트에 들어갈 기능 후보
- Main Workspace Room: 커밋 전 최종 확인 공간
- ReadMe 폴더: 팀원에게 남기는 규칙, 안내, 체크리스트
