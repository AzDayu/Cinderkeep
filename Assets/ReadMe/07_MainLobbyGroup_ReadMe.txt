MainLobbyGroup 안내

MainLobbyGroup은 프로젝트를 열었을 때 보이는 메인 허브 전용 그룹입니다.
팀원 작업 편의, 몰입감, 메뉴 이동을 위한 영역이며 실제 게임 로직을 넣는 곳이 아닙니다.

팀원분들은 MainLobbyGroup 내부 오브젝트를 수정하지 말아주세요.
허브 UI, BGM, 캐릭터 작업공간 연결은 팀장이 관리합니다.

실제 게임 작업 기준:
- Main_Lobby 씬에서는 최상위 Player 오브젝트만 게임 진입 표지로 사용합니다.
- Player를 누르면 Cinderkeep_Game 씬으로 이동합니다.
- 캐릭터 선택 창에서 본인 캐릭터를 클릭하면 선택 정보가 기억됩니다.
- Main_Lobby의 개인 작업 공간 버튼은 기억된 캐릭터의 개인 씬으로 자동 이동합니다.
- 잘못 선택했다면 캐릭터 상세 패널의 선택 취소 버튼으로 기억값을 지울 수 있습니다.
- 선택/취소 기록은 Assets/ReadMe/MainHub_CharacterSelection_Log.txt에 남습니다.
- 개인 작업 공간의 작업한 내용 폐기하기 버튼은 현재 씬을 처음 상태로 다시 엽니다.
- 이 버튼은 테스트 중 생긴 임시 상태를 버리는 용도이며, Git 커밋이나 파일 변경까지 되돌리지는 않습니다.
- 플레이어 조작, 몬스터, 자원, 건축, HUD 같은 실제 게임 오브젝트는 Cinderkeep_Game 씬에서 관리합니다.

작업 위치가 헷갈리면 먼저 개인 브랜치에서 작업하고, 머지 전에 팀장에게 공유해주세요.
