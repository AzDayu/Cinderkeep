1. 스샷 설명


1) StartPoint = 게임 실행시 시작되는 위치에서 촬영한 스크린샷입니다.

2) LongShot = 현재 설정된 맵에서 최대 거리로 설정된 모서리 부분(맵 끝)에서 촬영한 스크린샷입니다.

3) Jump_X = 점프 기능이 현재 제대로 작동하고 있는지에 대해서 여러가지로 작동해본 스크린샷입니다.

   3-1) Jump_0 = 점프하기 전 가장 작은 원형 발판 앞에 선 모습입니다.

   3-2) Jump_0_1 and 0_2 = 현재 가장 낮은 원형 발판 한정이긴 한데, 밀접한 상태에서 블록 방향으로 진행 시 플레이어 개체가 살짝 뜨는 현상이 나오고 있습니다.
                                      해당 상태에서 발판 방향으로 이동 시 살짝 유지되다가 블록에서 떨어지면 원래 높이로 돌아옵니다.
                                      현재 해당 높이의 블록이 원형 밖에 없어서 원형 발판이기에 이러한 현상이 나오는 것인지, 사각형 발판에는 이러한 현상이 나오지 않는 것인지는 아직 파악되지 않았습니다.
                                      게임 플레이에 있어서 크게 문제가 될 사항은 아니라고 판단되지만, 혹시 모르니 작성해두었습니다.
                                      0_1은 해당 상태를 촬영한 스크린샷이며, 0_2는 해당 상태보다 더 앞으로 나아갔음에도 고도가 낮아짐을 보여드리기 위해 촬영한 스크린샷입니다.

   3-3) Jump_1~3 = 1은 가장 낮은 원형 발판 위에서 촬영, 2는 중간 크기의 사각형 발판 위에서 촬영, 3은 가장 높은 사각형 기둥 앞에서 점프하며 촬영



2. 스샷 외 설명

1) WASD 이동은 정상적으로 작동하며, 점프 또한 정상적으로 작동합니다. 마우스를 이용한 시야 전환 또한 정상적으로 작동합니다.

2) 점프의 경우 입력시간의 길이에 상관없이 일정한 높이와 일정한 시간대로 점프하며, 점프가 끝나기 전에는 다시 점프가 입력되지 않습니다. 8방향 어느 쪽으로든 해당 방향으로 이동하면서 점프가 가능합니다. 단, 시야는 마우스 방향 그대로 따라갑니다.

3) 원형 발판은 밀접한 후 밀려나며 좌~우로 이동 가능. 사각형 발판은 밀접한 후 발판 방향으로 정면 이동 시 밀려나지 않으며, 좌 또는 우로 이동 시 해당 방향으로 미끄러짐.

4) 대각선 이동 = 가운데까지 대략 15초, 반대쪽 끝까지 대략 30초 (1초 오차 존재 가능)

5) 변을 따라 이동 = 가운데까지 대략 10초, 반대쪽 끝까지 대략 20초 (1~2초 정도 오차 존재 가능)

6) 시야 전환 속도의 경우, 몹의 배치나 자원 설정시 또 체감이 달라질 수 있겠지만 현재로서는 적당한 것 같습니다.
